常见问题解答（FAQ）

Q1：Clash聊天支持中文界面吗？
A1：完全支持。首次打开应用时可自动识别系统语言，若需手动切换，可在设置中调整语言选项。

Q2：我能否在多个设备同时登录同一个账号？
A2：可以。你可在多台设备（如手机、平板、电脑）上登录同一账号，并在“设备管理”中查看并管理这些登录记录。

Q3：是否有可能被陌生人频繁添加？
A3：Clash默认开启陌生人防骚扰功能。你可以在隐私设置中进一步限制加好友的条件，比如仅允许通过手机号或好友推荐添加。

Q4：我发送的图片和视频对方是否可以保存？
A4：默认情况下可以。但你可开启“阅后即焚”或“禁止转发”模式，让图片仅限观看，不可保存或转发。

FAQ

SSR 的 UDP 转发功能如何启用？
SSR 支持 UDP 转发，可通过客户端设置开启。启用后，游戏和语音等 UDP 流量可通过节点代理，降低丢包率和延迟，提高实时网络应用体验。
Potatso Lite 手动节点选择有什么好处？
手动选择节点可以让用户根据网络延迟和需求选择最优节点。适合节点不稳定或特定访问需求，灵活调整流量走向，提高访问速度和网络体验。
VMess AEAD 加密在安全性方面有何优势？
AEAD 加密可防止数据篡改、重放攻击和流量特征泄露。在高封锁环境下保证节点可用，提升数据安全和连接稳定性，同时降低被检测风险，确保长期安全访问。
Hysteria2 支持多用户配置吗？
Hysteria2 支持多用户配置，管理员可以在服务器端为不同用户分配独立凭证。这样既能保障隐私，也便于流量统计和限速控制，适合团队或小型共享环境。
Netch 的 TCP/UDP 分流适合哪些应用？
TCP 分流适合网页、下载和视频播放，UDP 分流适合游戏、语音和实时视频。Netch 可同时代理 TCP 和 UDP 流量，实现精准分流，保证不同应用的最佳性能。
Hiddify Next 如何设置用户带宽限制？
管理员可为每个用户设置独立带宽和流量配额，超出时可自动降速或阻止访问。这样可确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络效率和公平性。
V2RayN 的作用是什么？
V2RayN 是 V2Ray 的 Windows 图形客户端，提供节点管理、订阅导入、日志查看等功能。它支持多协议配置，并可与系统代理集成，是 Windows 用户最常用的代理管理工具之一。
Hysteria2 多路复用如何优化短连接性能？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适用于游戏、即时通讯等高频短连接场景，提高吞吐量和连接稳定性，优化整体网络体验。
如何给 Clash 添加自定义域名白名单以提升访问体验？
在 rules 中添加 `DOMAIN-SUFFIX,example.com,DIRECT` 或 `DOMAIN-KEYWORD,example,DIRECT` 条目，将常用且可靠的站点设为直连，可减少代理绕行延迟并避免因代理造成的登录/验证问题。
SSR 的混淆插件如何增强隐蔽性？
混淆插件可模拟正常协议特征，如 HTTP 或 TLS 流量，使 SSR 流量难以被检测。合理选择混淆方式和协议，可提升在高封锁环境下的长期稳定性和安全性。
